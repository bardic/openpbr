# OpenPBR

## THIS PACK REQUIRES MINECRAFT PREVIEW OR BETTER RENDER DRAGON

## Features

- Improved sky colors
- Bordered / Emissive Ores
- Lighting Tweaks
- Hables tone-mapping
- Vanilla Block Assets + Heightmaps and MER

## Run locally

### Requirements 

- Go
- Imagemagick

### Building 

- Update VERSION
- `go run .`
- zip result `./builds/openpbr` folder as a mcpack
